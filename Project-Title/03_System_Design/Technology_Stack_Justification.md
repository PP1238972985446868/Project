Technology Stack Justification สำหรับโปรเจคนี้

1.ภาษาโปรแกรม (Programming Language)

Python
เหตุผล:

เรียนรู้ง่าย โค้ดกระชับ เข้าใจง่ายสำหรับผู้เริ่มต้น

มีไลบรารีมาตรฐาน (random, string, tkinter) ที่รองรับการสร้างรหัสผ่านและ GUI ได้โดยไม่ต้องติดตั้งเพิ่ม

เหมาะกับโปรเจคที่ต้องการให้คนทั่วไปใช้งานง่ายและสามารถรันได้ทันที

2.Framework / Library

Tkinter (GUI Library)
เหตุผล:

ติดมากับ Python ใช้งานได้ทันที ไม่ต้องติดตั้งเพิ่มเติม

สร้าง GUI ได้รวดเร็วและเรียบง่าย

รองรับการออกแบบ UI ที่เข้าใจง่ายสำหรับผู้ใช้ทั่วไป (User-Friendly)

3.Database

ไม่มี Database ใช้
เหตุผล:

โครงการนี้ไม่จำเป็นต้องเก็บรหัสผ่านหรือข้อมูลผู้ใช้

ทำให้ระบบทำงานแบบ Standalone และง่ายต่อการใช้งาน

ลดความซับซ้อนและเพิ่มความปลอดภัย

สรุปเหตุผล:

Python + Tkinter: สร้างโปรแกรม GUI ที่รันง่าย, เรียนรู้เร็ว, UI เข้าใจง่าย
No Database: เหมาะกับโปรเจค Standalone เน้นการสร้างรหัสผ่านเฉพาะหน้าจอ ลดความซับซ้อน
เหมาะกับโจทย์นี้: เน้นสร้างรหัสผ่านแบบสุ่ม, ใช้งานง่าย, ไม่ต้องพึ่งเทคโนโลยีหรือฐานข้อมูลอื่น
