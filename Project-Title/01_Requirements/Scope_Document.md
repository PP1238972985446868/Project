Scope Document
Project Title: Random Password Generator System

1. วัตถุประสงค์ของโครงการ (Project Objectives)

ระบบสร้างรหัสผ่านสุ่ม (Random Password Generator) มีวัตถุประสงค์เพื่อสร้างรหัสผ่านที่มีความปลอดภัยสูงให้ผู้ใช้ โดยให้ผู้ใช้สามารถกำหนดเงื่อนไขต่าง ๆ ได้ เช่น ความยาวรหัสผ่าน การเลือกชนิดตัวอักษร (ตัวพิมพ์เล็ก, พิมพ์ใหญ่, ตัวเลข, อักขระพิเศษ) เพื่อช่วยเพิ่มความปลอดภัยในการใช้งานบัญชีต่าง ๆ

2. ขอบเขตของระบบ (System Scope)

อยู่ในขอบเขต (IN SCOPE)

1. ผู้ใช้สามารถกำหนดความยาวของรหัสผ่านได้
2. ผู้ใช้สามารถเลือกประเภทของตัวอักษรที่ต้องการให้รวมในรหัสผ่าน
3. ระบบสร้างรหัสผ่านแบบสุ่มตามเงื่อนไขที่ผู้ใช้เลือก
4. ระบบแสดงผลรหัสผ่านที่สร้างให้ผู้ใช้คัดลอกได้
5. ตรวจสอบความถูกต้องของ Input เช่น ความยาวต้องมากกว่า 0
6. UI แบบง่ายที่ให้ผู้ใช้กำหนดตัวเลือกได้

อยู่นอกขอบเขต (OUT OF SCOPE)

1. การบันทึกรหัสผ่านลงฐานข้อมูล
2. การเข้ารหัส (Encryption) ของรหัสผ่านที่สร้างขึ้น
3. การตรวจสอบระดับความแข็งแรงของรหัสผ่าน (Password Strength Meter)
4. การเชื่อมต่อกับระบบภายนอก เช่น Password Manager
5. การทำระบบ Login / Register ผู้ใช้
6. Multi-language interface

3. ผู้ใช้งานเป้าหมาย (Target Users)

* ผู้ใช้ทั่วไปที่ต้องการรหัสผ่านความปลอดภัยสูง
* นักเรียน/นักศึกษาในสาย IT ที่ใช้ระบบนี้เพื่อเรียนรู้หลักการสุ่มข้อความ
* ผู้ที่ต้องการเครื่องมือช่วยสร้างรหัสผ่านอย่างรวดเร็วและเชื่อถือได้

4. คุณสมบัติหลักของระบบ (Key Features)

* การสุ่มตัวอักษรตามตัวเลือกของผู้ใช้
* การสร้างรหัสผ่านที่มีความยาวตั้งแต่ X ถึง Y ตัวอักษร (กำหนดได้)
* UI ใช้งานง่าย
* แสดงผลรหัสผ่านให้คัดลอกได้

5. ข้อจำกัดของระบบ (System Constraints)

* ระบบต้องทำงานได้แบบ Stand-alone (ไม่มีฐานข้อมูล)
* ระบบต้องรันได้ในเครื่องผู้ใช้โดยไม่ต้องติดตั้งซอฟต์แวร์เพิ่มเติมมาก
* ใช้ภาษาโปรแกรมที่เข้าใจง่าย เช่น Python หรือ JavaScript

6. การยอมรับงาน (Acceptance Criteria)

1. ผู้ใช้ต้องสามารถสร้างรหัสผ่านตามความยาวที่ระบุ
2. ผู้ใช้สามารถเลือกประเภทตัวอักษรได้ครบทุกแบบ
3. ระบบต้องสร้างรหัสผ่านใหม่ทุกครั้งที่กดปุ่มสร้าง
4. รหัสผ่านต้องเปลี่ยนทุกครั้งและไม่น่าจะซ้ำกัน
5. UI ต้องเข้าใจง่าย ไม่ซับซ้อน
