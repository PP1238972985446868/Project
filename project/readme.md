# โปรแกรมสุ่มรหัสผ่าน (Random Password Generator)

สคริปต์ Python สำหรับสร้างรหัสผ่านที่ปลอดภัยและสุ่ม ตามระดับความซับซ้อนที่ผู้ใช้กำหนดเอง

## คุณสมบัติ (Features)

- **กำหนดความยาวได้เอง**: รองรับความยาวรหัสผ่านเท่าไหร่ก็ได้ (ขั้นต่ำ 4 ตัวอักษร)

- **ความซับซ้อนที่ยืดหยุ่น**: เลือกเปิด/ปิดการใช้งานของ:
  - ตัวอักษรพิมพ์ใหญ่ (`A-Z`)
  - ตัวอักษรพิมพ์เล็ก (`a-z`)
  - ตัวเลข (`0-9`)
  - สัญลักษณ์พิเศษ (`!@#$%...`)

- **รับประกันส่วนประกอบ**: ระบบจะตรวจสอบให้แน่ใจว่ามีตัวอักษรจากทุกหมวดหมู่ที่เลือกอย่างน้อย 1 ตัวในรหัสผ่านที่สร้างขึ้น

- **การสุ่มสมบูรณ์**: ใช้ฟังก์ชัน `random.shuffle` เพื่อป้องกันรูปแบบที่คาดเดาได้

## วิธีใช้งาน (Usage)

1.  **รันสคริปต์**:
    ```bash
    python test.py
    ```
    *(หมายเหตุ: หากคุณเปลี่ยนชื่อไฟล์สคริปต์ ให้ใช้ชื่อไฟล์นั้นแทน เช่น `python password_generator.py`)*

2.  **ทำตามคำแนะนำบนหน้าจอ**:
    - ระบุความยาวของรหัสผ่านที่ต้องการ
    - พิมพ์ `y` (ใช่) หรือ `n` (ไม่) เพื่อเลือกประเภทตัวอักษรที่ต้องการรวมในรหัสผ่าน
